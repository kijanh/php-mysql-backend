## Opis projekta:
Web aplikacija za upravljanje korisnicima (Users) koristeći PHP i MySQL bazu podataka.

Aplikacija implementira kompletan CRUD sistem:

- Create – dodavanje korisnika
- Read – prikaz svih korisnika
- Update – izmjena podataka korisnika
- Delete – brisanje korisnika

## Tehnologije:
- PHP (PDO)
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Pokretanje projekta:

1. Kreirati bazu `php_mysql_backend`
2. Importovati SQL tabelu `users`
3. Postaviti projekt u `htdocs`
4. Pokrenuti Apache i MySQL (XAMPP)
5. Otvoriti: http://localhost/php-mysql-backend
